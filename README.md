# 🩺 Simple Linux System Health Check Script

This project contains a basic yet effective **Bash script** to monitor the health of a Linux system. It checks key parameters such as:

- ✅ Disk Usage
- ✅ Memory Status
- ✅ CPU Load
- ✅ Logged-in Users
- ✅ Uptime
